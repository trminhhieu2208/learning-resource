# MARKDOWN CHEAT SHEET

## 1. CÚ PHÁP CƠ BẢN

| **Thành phần**        | **Cú pháp**      | **Mục đích / Ghi chú**                      |
| :-------------------- | :--------------- | :------------------------------------------ |
| **Tiêu đề (H1 - H6)** | `#` đến `######` | Phân cấp nội dung (Giảm dần từ 1 đến 6).    |
| **In đậm**            | `**Văn bản**`    | Nhấn mạnh nội dung quan trọng.              |
| **In nghiêng**        | `_Văn bản_`      | Nhấn mạnh nhẹ hoặc dùng cho thuật ngữ.      |
| **Đậm & Nghiêng**     | `**_Văn bản_**`  | Nhấn mạnh tối đa.                           |
| **Gạch ngang**        | `~~Văn bản~~`    | Đánh dấu nội dung đã thay đổi hoặc hủy bỏ.  |
| **Đường kẻ ngang**    | `---`            | Phân tách các phần lớn trong tài liệu.      |
| **Danh sách (KTT)**   | `-`              | Liệt kê các ý không phân cấp thứ tự.        |
| **Danh sách (CTT)**   | `1. 2. 3.`       | Liệt kê các bước hoặc thứ tự ưu tiên.       |
| **Trích dẫn**         | `> Văn bản`      | Trích dẫn lời nói hoặc đoạn văn tham chiếu. |
| **Liên kết**          | `[Tên](URL)`     | Dẫn nguồn đến trang web hoặc tài liệu khác. |
| **Hình ảnh**          | `![Alt](URL)`    | Chèn ảnh (cần có mô tả thay thế).           |
| **Mã nội tuyến**      | `` `code` ``     | Đánh dấu tên biến, hàm hoặc lệnh ngắn.      |

## 2. CÚ PHÁP MỞ RỘNG

| **Thành phần**           | **Cú pháp**          | **Chi tiết**                                   |
| :----------------------- | :------------------- | :--------------------------------------------- |
| **Bảng (Tables)**        | \`                   | Cột.                                           |
| **Khối mã (Code)**       | ` ```language `      | Hiển thị mã nguồn với màu cú pháp (Highlight). |
| **Task List**            | `- [ ]` hoặc `- [x]` | Quản lý tiến độ công việc.                     |
| **Chú thích (Footnote)** | `[^1]`               | Giải thích thuật ngữ ở cuối trang.             |
| **Chỉ số dưới**          | `H~2~O`              | Dùng cho công thức hóa học.                    |
| **Chỉ số trên**          | `X^2^`               | Dùng cho lũy thừa hoặc toán học.               |

## 3. QUY TẮC TRÌNH BÀY CHUYÊN NGHIỆP

| **Quy tắc**        | **Hành động**                     | **Lợi ích**                                    |
| :----------------- | :-------------------------------- | :--------------------------------------------- |
| **Ngắt dòng**      | Thêm 2 dấu cách ở cuối dòng.      | Xuống dòng mà không tạo đoạn mới.              |
| **Ký tự đặc biệt** | Sử dụng dấu gạch chéo `\`.        | Hiển thị các ký tự trùng với cú pháp Markdown. |
| **Tính nhất quán** | Chọn duy nhất `-` hoặc `*`.       | Đảm bảo thẩm mỹ và dễ bảo trì file.            |
| **Phân cấp**       | Tuân thủ thứ tự `H1 -> H2 -> H3`. | Tốt cho cấu trúc tài liệu và SEO.              |
| **Căn lề bảng**    | `:---`, `:---:`, `---:`.          | Tối ưu hóa hiển thị cho từng loại dữ liệu.     |

# Learning Resource

Đây là repository mình dùng để lưu trữ tài liệu và ghi chép trong quá trình học lập trình.

## Lưu ý

Các nội dung trong repo được ghi lại theo hiểu biết và góc nhìn cá nhân của mình ở từng thời điểm học, nên có thể còn thiếu sót hoặc chưa hoàn toàn chính xác.

Vì vậy, hãy xem đây là tài liệu **tham khảo** và nên đối chiếu thêm với tài liệu chính thống khi cần.
